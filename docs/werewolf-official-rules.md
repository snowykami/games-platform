# 狼人杀官方规则参考

最后核查日期：2026-06-21

## 适用范围

所有叫 Werewolf / 狼人杀的游戏，并不存在一个全球唯一的“官方角色表”。本项目后续建议按下面优先级建立规则口径：

1. **经典官方基准**：Zygomatic / Asmodee 的 *The Werewolves of Miller's Hollow*（米勒山谷狼人）官方规则书。
2. **西方大角色池参考**：Bezier Games 的 *Ultimate Werewolf* / *Ultimate Werewolf Extreme* 官方产品说明和配套生态。
3. **中文常用桌游/网杀标准**：预女猎白、预女猎守等本土常见版型。它们应该作为单独的变体模块，而不是直接混入米勒山谷基础规则。

当不同来源规则不一致时，必须在房间预设里明确标注使用的是哪一套变体。

## 当前项目角色数量

项目当前已实现的狼人杀角色为 **7 个**：

- Villager / 村民
- Werewolf / 狼人
- Seer / 预言家
- Guard / 守卫
- Witch / 女巫
- Hunter / 猎人
- Idiot / 白痴

当前角色池更接近中文常见的“预女猎白 / 守卫”版型，而不是米勒山谷基础盒。

## 为什么有些资料会说有 40-100+ 个角色

主人提到的“四五十种角色”是存在的，但主要来自其他官方产品线或扩展合集，而不是米勒山谷基础盒。

常见的大角色池来源：

- **Bezier Games 的 Ultimate Werewolf**：出版方介绍入门版本包含 **34 张角色牌**。
- **Ultimate Werewolf: Ultimate Edition / Deluxe 时代版本**：常见资料会提到 **40+ 个独特角色**。
- **Ultimate Werewolf Extreme**：新一代大规模版本，面向 5-75 人局。官方产品页说明它是面向超大局的 Ultimate Werewolf 产品，二手/评测资料常列出 **88 张角色牌**。
- **The Werewolves of Miller's Hollow: Characters** 扩展：Asmodee 商品页说明它新增 **16 个角色**。
- 米勒山谷产品线本身还有多款扩展。如果把基础盒、扩展、事件牌、建筑牌、状态牌一起统计，可用角色/状态数量会明显超过基础盒的 8-9 种。

因此，讨论“角色数量”时必须同时说明来源口径：

- **米勒山谷基础官方版**：8 种发牌角色；如果把警长状态也算上，则是 9 种角色/状态。
- **米勒山谷 + Characters 扩展**：基础角色再加 16 个新角色。
- **Ultimate Werewolf 系列**：几十种角色，不同版本常见口径包括 34、40+、88 张角色牌。
- **中文线上狼人杀**：通常包含大量本土角色和房规变体，应作为独立预设体系。

## 米勒山谷官方基础版

官方产品信息：

- Zygomatic 官方页面将 *The Werewolves of Miller's Hollow* 标为 8-18 人、约 30 分钟、10 岁以上的社交推理派对游戏，组件包含 24 张牌和 1 本规则书。
- Asmodee USA 商品页也列出同样的 8-18 人范围，并标注出版方为 Zygomatic。
- Zygomatic 官方规则书列出的 24 张牌如下：
  - 4 张 Werewolves / 狼人
  - 13 张 Simple Villagers / 普通村民
  - 1 张 Seer / 预言家
  - 1 张 Hunter / 猎人
  - 1 张 Witch / 女巫
  - 1 张 Thief / 盗贼
  - 1 张 Cupid / 丘比特
  - 1 张 Little Girl / 小女孩
  - 1 张 Sheriff / 警长牌或警长状态

如果只统计“会作为隐藏身份发给玩家的角色”，基础版有 **8 种角色**：

- Werewolf / 狼人
- Simple Villager / 普通村民
- Seer / 预言家
- Hunter / 猎人
- Witch / 女巫
- Thief / 盗贼
- Cupid / 丘比特
- Little Girl / 小女孩

如果把 Sheriff / 警长作为公开状态或特殊牌也算进去，基础盒有 **9 种角色/状态**。

注意：警长不是像普通隐藏身份一样发给玩家的角色牌，而是在游戏过程中选举产生的公开状态。米勒山谷规则书中，警长的投票权重为双倍。

## 米勒山谷官方流程

### 胜利目标

- 村民阵营胜利：消灭所有狼人，并且至少还有一名村民存活。
- 狼人阵营胜利：在至少一名狼人存活的情况下，最后一名村民死亡。
- 如果存在一对“村民 / 狼人”情侣，则这对情侣拥有特殊胜利条件：他们需要共同存活，并消灭其他所有玩家。

### 准备阶段

- 选择一名主持人。
- 主持人选择本局使用的身份牌组合，洗牌，并给每名玩家发一张隐藏身份牌。
- 玩家秘密查看自己的身份。
- 村庄进入夜晚。

如果启用了相关角色，则有可选的首夜准备流程：

1. 盗贼醒来，可以从两张额外的中央身份牌中选择是否交换。如果两张中央牌都是狼人，盗贼必须拿走一张狼人牌。
2. 丘比特醒来，指定两名玩家成为情侣。
3. 情侣醒来，相互确认身份，然后重新入睡。

### 标准夜晚

米勒山谷基础规则书中的标准夜晚顺序：

1. 预言家醒来，查验一名玩家的真实身份。
2. 狼人醒来，互相确认，私下协商，并选择一名受害者。
3. 女巫醒来，主持人告知狼人最新选择的受害者；女巫可以根据药水剩余情况选择使用解药和/或毒药。
4. 村庄醒来，主持人公布夜晚结果。

官方规则说明：如果狼人无法就受害者达成一致，则当晚没有狼刀出局。

### 白天

1. 主持人公布狼人夜晚选择的受害者。
2. 如果受害者是猎人，猎人可以在死亡前发动反击。
3. 如果一名情侣死亡，另一名情侣也立刻殉情死亡。
4. 存活玩家进行讨论。
5. 村庄同时投票。
6. 得票最多的玩家被放逐。
7. 如果投票平票，则无人被放逐。
8. 被淘汰玩家公开自己的身份牌，并且不能再参与交流。
9. 存活玩家重新入睡，进入下一夜。

## 中文常见变体说明

中文狼人杀经常使用米勒山谷基础盒里没有的角色和规则。

“预女猎白 / 守卫”类常见角色：

- 平民 / Villager
- 狼人 / Werewolf
- 预言家 / Seer
- 女巫 / Witch
- 猎人 / Hunter
- 白痴 / Idiot
- 守卫 / Guard
- 警长 / Sheriff，通常是选举产生的徽章/状态，不是隐藏身份

中文守卫版型常见夜晚顺序：

1. 守卫行动。
2. 预言家查验。
3. 狼人协商并刀人。
4. 女巫得知狼刀目标，并选择是否用药。
5. 猎人 / 白痴是否需要首夜确认，或者只在死亡时确认，取决于具体房规。

常见中文规则差异，后续应在每个预设里显式配置：

- 守卫通常不能连续两晚守护同一名玩家。
- 守卫通常可以守自己。
- 女巫能否自救取决于房规。
- 守卫和女巫同救同一目标时，许多中文规则会导致目标死亡，即“同守同救 / 奶穿”；这不是米勒山谷基础规则。
- 猎人被女巫毒死时通常不能开枪，但不同桌规可能不同。
- 狼人自刀在中文玩法中通常允许。
- 中文玩法中警长票权常见为 1.5 票；米勒山谷规则为 2 票。
- 中文狼人杀常见胜利条件是“屠边”：狼人消灭所有村民或所有神职即胜。米勒山谷基础规则更接近“最后一名村民死亡”；当前项目使用的是简化数值条件，后续需要按预设重新梳理。

## 本项目后续实现建议

### 基础官方预设

新增一个 “Miller's Hollow Classic / 米勒山谷经典” 预设，适配 8-18 人：

- 狼人数量按官方建议表：
  - 8-11 人：2 狼
  - 12-17 人：3 狼
  - 18 人：4 狼
- 1 名预言家
- 剩余玩家为普通村民

这个预设应只使用标准夜晚流程，不启用盗贼、丘比特等准备阶段角色。

### 官方基础扩展角色

在继续加入更多中文自定义角色前，建议先补齐米勒山谷基础盒缺失角色：

- 盗贼 / Thief
- 丘比特 / Cupid
- 情侣状态 / Lovers
- 小女孩 / Little Girl
- 警长选举与警长状态 / Sheriff

### 大角色池路线

如果项目想实现“四五十种角色”的体验，建议单独建立 **Ultimate 风格大角色池**，不要把它混进米勒山谷基础规则里。

推荐实现阶段：

1. 保持当前中文快速预设稳定。
2. 补齐米勒山谷基础盒缺失角色：盗贼、丘比特、情侣、小女孩、警长。
3. 将米勒山谷 Characters 扩展作为具名扩展模块加入。
4. 按机制分批加入 Ultimate 风格角色：
   - 信息类角色：Aura Seer、Apprentice Seer、Beholder、Paranormal Investigator。
   - 保护类角色：Bodyguard、Diseased、Tough Guy。
   - 击杀类角色：女巫变体、猎人变体、Village Idiot 类干扰角色。
   - 狼队角色：Minion、Dream Wolf、Wolf Cub、Sorcerer 类角色。
   - 中立角色：Tanner、Cursed、Doppelganger 类变体。

每一批角色都需要来源说明和平衡元数据。不要把所有角色静默混入默认预设。

### 中文变体预设

保留当前 6-12 人快速预设，但应标注为中文常见变体，而不是米勒山谷官方基础规则。

对于“预女猎白 / 守卫”预设，建议文档化并实现以下开关：

- 守卫不能连续两晚守护同一目标。
- 守卫允许自守。
- 女巫是否允许自救：可配置。
- 守卫和女巫同目标时是否奶穿：可配置；中文预设默认可启用奶穿。
- 猎人被女巫毒死是否不能开枪：可配置。
- 警长启用 / 禁用。
- 胜利条件：屠边 vs 数值平衡。

### 当前缺口

- 缺少米勒山谷基础角色：盗贼、丘比特、情侣、小女孩、警长。
- 尚未实现守卫不能连续守同一目标。
- 尚未实现警长选举和警长票权。
- 当前胜利条件需要按不同预设重新校准。
- 白天投票平票无人出局已经符合米勒山谷规则。
- 狼队夜晚共识应支持官方“无法决定则无刀”的结果；当前交互实现是要求狼人达成一致或全员弃权。
